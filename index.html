Nokia Internships

Two full-time positions at Nokia

I've worked 2 different positions with Nokia. In Summer 2020, I interned as a Software Test Engineer. Here are some things I got to do:

  • Automated GUI testing using Selenium, Node.js, and the webdriverio API
  • Developed a scrolling function to improve outdated API commands in GUI testing
  • Identified various major bugs in software before quarterly product release

The other position I worked was as Lab Technician Co-op. This was more focused on hardware and below are some of the tasks I performed:

  • Managed customer modem racks using fiber optics and DSL
  • Assembled and tested terabit internet routers as well as several computer systems to be used by Nokia employees with varying needs and specifications

Raspberry Pi 4 Security Camera

Using OpenCV 4.0 and Pi Camera

This project was really exciting. After purchasing a Raspberry Pi 4 and a Pi Camera module, I quickly tried to install OpenCV to build my secrity camera. Turns out, it was a difficult process for a complete beginner with lacking knowledge in CLI commands.

After quite a bit of effort, I overcame my technical difficulties and I was able to get it set up. After integrating some Python scripts, I was able to get live video footage on a web server, as well as automatic emails with security updates when a person is detected.

Live Weather App

Using Weatherstack API

This project was my first real attempt at understanding APIs and trying to get into the world of open source projects. I followed a Udemy course that gave me all the right information to create my own live weather app.

Not only was this a great experience for learning about APIs (which I was also doing in my Nokia job) but I also learned about deploying apps with Heroku - a very useful skill.
