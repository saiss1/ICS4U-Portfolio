The Game of Life

This challege was by far my favourite part of this course. It took elements from learning and creativity and made a great balance by putting them together. What I really enjoyed was learning about all these new keywords and learning about Object oriented programming. The best part was that I had the ability to refactor the program anyway I see fit. The Game of Life is a very interesting algorithmic program that combines mathematical concepts, computer science concepts, and visually satisfying patterns. Overall, a fun time.
